jQuery Parallax Plugin

わりと簡単にパララックスさせられます。

一番簡単な使い方

JavaScipt2つとCSSを1つ読み込んでください。

	<link rel="stylesheet" href="path/to/jquery.loadprogress.css" />
	<script type="text/javascript" src="path/to/jquery.loadprogress.js"></script>
	<script type="text/javascript" src="path/to/jquery.parallax.js"></script>

動きは

$('#hoge').css();
で指定できるプロパティが指定できます。

使い方はまず

		$.parallax.init();
を呼び出します。オプションについては後述します。

次にスクロールに合わせてアニメーションさせたい要素で関数を呼びます。

		$('#hoge')
			.paraAnimate({
				startProp:{
					height:0
				},
				easing:'easeOutBounce'
			});

最後に

$.parallax.start();
を呼び出します。

たったこれだけでアニメーションさせることができます！

詳細な使い方

$.parallax.init();に渡すことができるプロパティの一覧と解説です。

useLoadprogress Boolean (true false) true trueの場合、jquery-loadprogress-pluginを使用します。
使用しな場合、haedタグ内で、jquery.loadprogress.jsとjquery.loadprogress.cssを読み込む必要はありません。
また、（このHTMLのように）imgが一枚もない場合はfalseにしてください。（むしろ必要ないでしょう。）
onScroll Function なし（undefined） 関数を渡すと、スクロールイベント発生時に実行されます。
keyScrollAmountth> Number 20 キーボードの上下キーで移動する際のピクセル数です。
scrollRate Number 0.8 マウスホイールでスクロールさせた時の移動量の調整用です。
0に近づくほど遅くなり、1を超えると速くなります。
startPosAutoOffset Number 0 start:'auto'にした際のオフセット値です。
endPosAutoOffset Number -100 end:'auto'にした際のオフセット値です。
mode String ('scroll' 'page') 'scroll' 未だ使われていません！

$('#hoge').paraAnimate()に渡す変数のプロパティ一覧です。

start Number String ('auto') 'auto' アニメーションを開始するピクセル数を設定してください。
また'auto'を渡すと、アニメーション対象の要素の
ドキュメントに対する位置+ウィンドウの高さ+前述のstartPosAutoOffsetの値がセットされます。
end Number String ('auto') 'auto' アニメーションを終了するピクセル数を設定してください。
また'auto'を渡すと、アニメーション対象の要素の
ドキュメントに対する位置+前述のendPosAutoOffsetの値がセットされます。
easing String (ex 'swing') 'swing' $('#hoge').animate()などと同じイージング名を設定してください。
jquery.easing.jsを利用することもできます！（その際は、このプラグインより前に読み込んでください。）
startProp Object なし（undefined） アニメーション開始時のCSSプロパティを渡してください。
省略することも可能です。省略した場合は、endPropに渡されたCSSプロパティを元に
CSSで設定されている値が自動的にセットされます。
$('#hoge').css()と同じようにCSSプロパティ名のオブジェクトを渡してください。 
					$('#hoge').paraAnimate({
						startProp:{
							height:0,
							opacity:0
							marginLeft:-500
						}
					})
また、heightとwidthに関しては明示的に'auto'を渡すこともできます。 この場合、アニメーション開始前に自動的に高さを取得します。
endProp Object なし（undefined） アニメーション終了時のCSSプロパティを渡してください。
省略することも可能です。省略した場合は、startPropに渡されたCSSプロパティを元に
CSSで設定されている値が自動的にセットされます。
$('#hoge').css()と同じようにCSSプロパティ名のオブジェクトを渡してください。 
					$('#hoge').paraAnimate({
						endProp:{
							height:100,
							opacity:1
							marginLeft:0
						}
					})
また、heightとwidthに関しては明示的に'auto'を渡すこともできます。 この場合、アニメーション開始前に自動的に高さを取得します。

全てのプロパティに記述した場合は以下のようになると思います。

		$('#hoge')
			.pareAnimate({
				start:400,
				end:600,
				easing:'swing'
				startProp:{
					height:0,
					opacity:0
					marginLeft:-500
				},
				endProp:{
					height:100,
					opacity:1
					marginLeft:0
				}
			});
